## 🚀 ขั้นตอนการติดตั้ง (สำคัญ!)

### ขั้นตอนที่ 1: สร้างฐานข้อมูล
1. เปิด XAMPP Control Panel
2. Start Apache และ MySQL
3. เปิดเว็บเบราว์เซอร์ไปที่ `http://localhost/phpmyadmin`
4. คลิก "New" เพื่อสร้างฐานข้อมูลใหม่
5. ตั้งชื่อฐานข้อมูล: `eexam_db`
6. เลือก Collation: `utf8mb4_unicode_ci`
7. คลิก "Create"
8. เลือกฐานข้อมูล `eexam_db`
9. คลิกแท็บ "Import"
10. เลือกไฟล์ `C:\xampp\htdocs\exam\database.sql`
11. คลิก "Go"
12. รอจนกว่าจะแสดงข้อความ "Import has been successfully finished"

### ขั้นตอนที่ 2: ทดสอบระบบ
1. เปิดเว็บเบราว์เซอร์
2. ไปที่ `http://localhost/exam/`
3. คุณจะเห็นหน้าหลักสำหรับนักเรียนกรอกรหัส

### ขั้นตอนที่ 3: เข้าสู่ระบบ Admin
1. คลิกปุ่ม "เข้าสู่ระบบ Admin" ที่มุมขวาบน
2. หรือไปที่ `http://localhost/exam/admin/login.php`
3. ใช้ข้อมูลเข้าสู่ระบบ:
   - **Username:** `admin`
   - **Password:** `admin123`
4. คลิก "เข้าสู่ระบบ"

## 📋 สิ่งที่ต้องทำต่อไป

### ไฟล์ที่ยังต้องสร้าง (จำเป็นสำหรับระบบเต็มรูปแบบ)

#### 1. ระบบจัดการนักเรียน
- `admin/students.php` - หน้าจัดการนักเรียน
- `admin/students_process.php` - ประมวลผลนักเรียน
- `admin/students_import.php` - นำเข้านักเรียนจาก Excel

#### 2. ระบบจัดการข้อสอบ
- `admin/questions.php` - หน้าจัดการข้อสอบ
- `admin/questions_process.php` - ประมวลผลข้อสอบ
- `admin/questions_upload.php` - อัพโหลดข้อสอบ
- `admin/questions_import.php` - นำเข้าข้อสอบจาก Excel

#### 3. ระบบสอบสำหรับนักเรียน
- `student/exam_instructions.php` - คำชี้แจงก่อนสอบ
- `student/exam.php` - หน้าสอบ
- `student/exam_process.php` - ประมวลผลการสอบ
- `student/exam_result.php` - แสดงผลการสอบ

#### 4. ระบบติดตามและรายงาน
- `admin/exam_monitor.php` - ติดตามการสอบ
- `admin/reports.php` - รายงานผลการสอบ
- `admin/reports_export.php` - ส่งออกรายงาน

#### 5. ระบบตั้งค่า
- `admin/settings.php` - ตั้งค่าระบบ
- `admin/settings_process.php` - ประมวลผลการตั้งค่า

#### 6. โฟลเดอร์สำหรับอัพโหลด
- `uploads/questions/` - เก็บรูปภาพข้อสอบ
- `uploads/temp/` - ไฟล์ชั่วคราว
