## การติดตั้งระบบ

### ความต้องการของระบบ
- PHP 7.4 หรือสูงกว่า
- MySQL 5.7 หรือสูงกว่า
- Apache/Nginx Web Server
- XAMPP/WAMP/LAMP (แนะนำ)

### ขั้นตอนการติดตั้ง

#### 1. ติดตั้งฐานข้อมูล
```bash
# เปิด phpMyAdmin หรือ MySQL Command Line
# สร้างฐานข้อมูลโดยใช้ไฟล์ database.sql
```

1. เปิด phpMyAdmin (http://localhost/phpmyadmin)
2. คลิก "New" เพื่อสร้างฐานข้อมูลใหม่
3. ตั้งชื่อฐานข้อมูลว่า `eexam_db`
4. เลือก Collation เป็น `utf8mb4_unicode_ci`
5. คลิก "Create"
6. เลือกฐานข้อมูล `eexam_db` ที่สร้างขึ้น
7. คลิกแท็บ "Import"
8. เลือกไฟล์ `database.sql`
9. คลิก "Go"

#### 2. ตั้งค่าการเชื่อมต่อฐานข้อมูล
แก้ไขไฟล์ `config/database.php` (ถ้าจำเป็น):
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'eexam_db');
```

#### 3. เข้าใช้งานระบบ

**สำหรับ Admin:**
- URL: `http://localhost/exam/admin/login.php`
- Username: `admin`
- Password: `admin123`

**สำหรับนักเรียน:**
- URL: `http://localhost/exam/`
- กรอกรหัสนักเรียนเพื่อเข้าสอบ

## การคำนวณคะแนน

### สูตรการคำนวณ
- **คะแนนรวม** = จำนวนข้อที่ถูก × คะแนนต่อข้อ
- **เปอร์เซ็นต์** = (คะแนนที่ได้ / คะแนนเต็ม) × 100

### ตัวอย่าง
- ข้อสอบ 40 ข้อ เก็บ 20 คะแนน
- คะแนนต่อข้อ = 20 / 40 = 0.5 คะแนน
- ถ้าทำถูก 30 ข้อ = 30 × 0.5 = 15 คะแนน
- เปอร์เซ็นต์ = (15 / 20) × 100 = 75%

## ค่าทางสถิติ

### Mean (ค่าเฉลี่ย)
```
Mean = ΣX / N
```
โดย X = คะแนนของนักเรียนแต่ละคน, N = จำนวนนักเรียน

### SD (ส่วนเบี่ยงเบนมาตรฐาน)
```
SD = √[Σ(X - Mean)² / N]
```

### P-Value (ค่าความยากง่าย)
```
P = (จำนวนคนที่ตอบถูก / จำนวนคนทั้งหมด)
```
- P > 0.70 = ข้อง่าย
- 0.30 ≤ P ≤ 0.70 = ข้อปานกลาง
- P < 0.30 = ข้อยาก

### R-Value (ค่าอำนาจจำแนก)
```
R = (PH - PL)
```
โดย PH = สัดส่วนกลุ่มสูงที่ตอบถูก, PL = สัดส่วนกลุ่มต่ำที่ตอบถูก
- R ≥ 0.40 = ดีมาก
- 0.30 ≤ R < 0.40 = ดี
- 0.20 ≤ R < 0.30 = พอใช้
- R < 0.20 = ควรปรับปรุง

## การแก้ปัญหา

### ไม่สามารถเข้าสู่ระบบ Admin ได้
1. ตรวจสอบว่าฐานข้อมูลถูกสร้างแล้ว
2. ตรวจสอบ username และ password
3. ลองรีเซ็ตรหัสผ่านในฐานข้อมูล

### ไม่พบรหัสนักเรียน
1. ตรวจสอบว่ามีข้อมูลนักเรียนในระบบแล้ว
2. ตรวจสอบรหัสนักเรียนให้ถูกต้อง
3. เพิ่มข้อมูลนักเรียนในหน้า Admin

### ข้อสอบไม่แสดง
1. ตรวจสอบว่ามีข้อสอบในชุดข้อสอบแล้ว
2. ตรวจสอบว่าเปิดใช้งานชุดข้อสอบแล้ว
3. ตรวจสอบวันเวลาการสอบ

### รูปภาพไม่แสดง
1. ตรวจสอบว่าอัพโหลดรูปภาพแล้ว
2. ตรวจสอบ path ของรูปภาพ
3. ตรวจสอบสิทธิ์ folder uploads/
